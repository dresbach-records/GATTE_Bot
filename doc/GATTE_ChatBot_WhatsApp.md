4. Fluxo de Atendimento

01	Cliente envia mensagem
Mensagem recebida via WhatsApp. Webhook captura e encaminha ao servidor em tempo real.
02	Identificacao do cliente
Sistema verifica se o cliente ja existe na base. Se sim, carrega historico e dados do perfil.
03	Exibicao do menu ou processamento livre
Se inicio de conversa ou comando 'menu': exibe menu estruturado. Se mensagem livre: processa com IA.
04	Processamento com IA
Mensagem + historico + instrucoes da GATTE sao enviados para a API de IA. A IA gera resposta contextualizada.
05	Acao especifica (se necessario)
Se detectado chamado: gera ticket. Se captacao de lead: salva em CRM. Se agendamento: cria compromisso.
06	Resposta enviada ao cliente
Servidor envia a resposta via API do WhatsApp. Historico e salvo para proxima interacao.
07	Escalonamento humano
Se complexidade alta ou pedido explicito: transfere para atendente com todo o contexto ja registrado.
